PROJECTS

TwoDoor

| JavaScript, JQuery, HTML & CSS, Firebase, D3 |

TwoDoor is an adaptive Flashcard Tutoring app created in just two weeks using agile development. What makes it unique is its intelligent card selection algorithm, which is an implementation of the Lietner system. We achieved user authentication and a database, so the user can create their own persisting custom decks. We used Firebase for its compatability with various platforms and lightweight interface.

Once we had a working backend, we moved to implementing Holberton's API into our app. The API gave us access to our peer's names and faces, which we used to render decks based on cohort. We marketed this feature as a way to remember our peers from across the globe.
